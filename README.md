# California_Sugar_Tax_SU

This is my final repository for the California Sugar Tax research project I worked on between 2024 and 2025. The repository here is a collection of my contributions to the project with data collection, manipulation, and analysis. It is a final collection of publicly available data from UCLA's Center for Health Policy Research, with manipulated datasets created in Excel and Python. A range of demographic data, including obesity rates, race and ethnicity, employment, and insurance type, was collected for adults across all California counties to inform our simulations. 

In this project, I collaborated with Dr. Elio Zhang to develop an ML model for working with ill-shaped panel data using cross validation in an ensemble setting. The methods we used incorporated multiplicative noise to improve results compared to the estimators on their own, yielding an 8% reduction in prediction error, which can be used in a variety of settings. I analyzed the results, comparing our simulated results to the observed results to see how obesity rates would have changed in our simulated San Francisco County.
